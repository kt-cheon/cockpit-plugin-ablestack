HCI 공유 파일 구성 마법사

HCI 공유 파일을 모든 호스트에 구성하기 위해, 공유 파일 배포를 단계별로 체계적으로 진행합니다.

HCI 공유 파일 구성 마법사는 내부 Glue 스토리지를 기반으로 RBD 이미지를 생성하고, 이를 하나의 공유 파일 환경으로 구성하며, 구성에 필요한 설정을 자동으로 생성하는 도구입니다. HCI 공유 파일 구성을 위해서는 다음 정보가 필요합니다.

  • RBD 이미지 생성에 필요한 용량 설정
  • 공유 파일로 구성할 생성된 RBD 이미지 선택
  • 호스트별 원격 관리(IPMI) 정보

필요한 정보를 먼저 준비하십시오. 정보가 준비되었다면 "다음" 버튼을 눌러 HCI 공유 파일 구성을 시작합니다.

HCI 공유 파일을 구성하기 전, 먼저 공유 파일로 사용할 이미지에 필요한 용량을 설정합니다.

GB

Info alert:HCI 공유 파일 구성 시 참고사항

공유 파일 구성을 진행하기 전에, 내부 Glue 스토리지에서 공유 파일로 사용할 RBD 이미지를 먼저 생성해야 합니다.

설정한 공유 파일 용량을 기준으로 RBD 이미지를 생성하며, 생성이 완료된 후 다음 단계로 진행해 주시기 바랍니다.

RBD 이미지는 이미지 1개당 최대 2TB까지 생성되며, 공유 파일 용량이 2TB를 초과하는 경우 용량에 맞게 여러 개의 RBD 이미지로 분할되어 생성됩니다.

2TB를 초과하는 용량을 설정하신 경우에는 분할 생성된 RBD 이미지가 모두 준비되었는지 확인한 후 다음 단계로 진행해 주시기 바랍니다.

이미 RBD 이미지가 생성되어 있는 환경에서 다시 구성을 진행하는 경우, 기존에 생성된 RBD 이미지를 선택하거나 필요 시 새로 생성한 뒤 다음 단계로 진행해 주시기 바랍니다.

HCI 공유 파일 전용 디스크로 관리할 'RBD 이미지'를 선택해야 합니다.


Info alert:HCI 공유 파일 전용 디스크 자원 구성 시 참고사항
