로컬 스토리지 구성 마법사

로컬 스토리지를 모든 호스트에 구성하기 위해, 스토리지 배포를 단계별로 체계적으로 진행합니다.

로컬 스토리지 구성 마법사는 단일 서버의 로컬 스토리지를 활용하여 운영에 필요한 설정을 자동화 하는 도구입니다. 로컬 스토리지를 구성하기 위해서는 다음과 같은 정보가 필요합니다.

  • 가상화 영역 디스크의 RAID 구성(초기 BIOS에서 설정) - 환경에 따라 적절한 RAID 타입 설정
  • 운영 안정성을 뒷받침하는 적정 디스크 용량 마련

위 정보를 미리 준비한 후, "다음" 버튼을 눌러 로컬 스토리지 구성을 시작하십시오.

로컬 스토리지 구성 시, 서버에서 물리적으로 인식된 디스크를 선택하여 관리 대상으로 지정해야 합니다.


Info alert:로컬용 디스크 자원 구성 시 참고사항

로컬 스토리지는 서버 내부의 직접 연결 디스크를 기반으로 동작하며, 운영 환경에 따라 안정성과 성능에 큰 영향을 미칩니다. 구성 시에는 디스크의 종류, 용량, 연결 경로를 반드시 확인하시기 바랍니다.
